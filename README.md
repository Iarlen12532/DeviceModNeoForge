# MrCrayfish's Device Mod - NeoForge 1.21.1

Recriação do mod **MrCrayfish's Device Mod** para **Minecraft 1.21.1** com **NeoForge**.

## Funcionalidades

- **Laptop** com Sistema Operacional próprio
- **App Store** - Baixe e instale apps programáveis
- **WiFi** - Sinal diminui com a distância do router
- **Ethernet** - Conexão cabeada com velocidade constante
- **Impressora** funcional com papel e tinta
- **Flash Drive** - Armazenamento portátil
- **16 Cores** para laptop, printer, router, flash drive, office chair

## Como usar as texturas do mod original

1. Extraia o JAR original do Device Mod
2. Copie as pastas:
   - `assets/cdm/textures/items/` → `src/main/resources/assets/devicemod/textures/items/`
   - `assets/cdm/textures/model/` → `src/main/resources/assets/devicemod/textures/model/`
   - `assets/cdm/textures/gui/` → `src/main/resources/assets/devicemod/textures/gui/`
   - `assets/cdm/textures/app/` → `src/main/resources/assets/devicemod/textures/app/`
3. Substitua os placeholders pelas texturas reais

## Compilar

### GitHub Actions (Recomendado para celular)
1. Faça push para o GitHub
2. Vá em Actions → Build Device Mod
3. Baixe o artifact "DeviceMod-Build"

## Requisitos
- Minecraft 1.21.1
- NeoForge 21.1.0+
- Java 21
